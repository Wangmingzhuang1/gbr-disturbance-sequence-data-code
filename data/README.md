# Data folder

This folder contains CSV source and lookup data used by the analysis pipeline.

Included CSV groups:

- `IDCKMSTM0S.csv`: tropical cyclone archive CSV used by the storm-exposure builder.
- `daily_climate_full/`: daily thermal-stress CSV files used to summarize DHW by AIMS ecological year.
- `reef_raw/`: ecological monitoring CSV files used to merge hard coral, algae, juvenile, herbivore, and COTS fields.
- `sites_decimal.csv`, `sites_lon_lat.csv`, and related lookup CSV files.

Non-CSV spatial boundary files used only for map rendering are not included here. Their source links are listed in `../DATA_AVAILABILITY.md`.
